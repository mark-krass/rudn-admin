## Запуск приложения
Перейти в консоли в корневой каталог приложения.<br>
Выполнить сборку:<br>
`gradle clean build -x test`<br>
Собрать docker image на Linux:<br>
`bash docker-build.sh`<br>
или если Windows:<br>
`.\docker-build.ps1`<br>
Запустить docker-контейнеры:<br>
`docker compose up -d`<br>
Для остановки приложения нужно выполнить команду:<br>
`docker compose down`<br>
